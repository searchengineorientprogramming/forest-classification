# Cross-correlation signal machine learning

This folder contains cross-correlation seismic data, code to calculate the average geometric phase change.

## Contents

- `CC_for_ZZ_nonforest`, `CC_for_ZZ_forest` - The cross-correlation seismic data of DPZ component.  

- `Average_phase_change_CC_ZZ.ipynb` - The code to calculate the average geometric phase change
- `utils.py` - Related functions used in `Average_phase_change_CC_ZZ.ipynb`.

- `results` - The folder save the output figures.


## Usage
Directly run `Average_phase_change_CC_ZZ.ipynb` to calculate.
